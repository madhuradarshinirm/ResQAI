# 🚨 ResQAI – AI-Powered Disaster Management Platform

ResQAI is an AI-powered disaster management platform designed to help citizens and disaster management authorities respond quickly and efficiently during emergencies. The platform combines real-time weather monitoring, AI-based disaster prediction, interactive disaster maps, SOS reporting, shelter location, community reporting, and an intelligent AI Assistant into one unified system.

---

## 🌟 Features

### 👤 Citizen Portal
- 🌦 Live Weather Monitoring
- 🤖 AI-Based Disaster Risk Prediction
- 🗺 Interactive Disaster Map
- 🚨 Emergency SOS Requests
- 🏠 Shelter Finder
- 💬 AI Assistant (Google Gemini)
- 📝 Community Incident Reporting

### 🛡 Admin Portal
- 📊 Disaster Analytics Dashboard
- 🚨 Monitor SOS Requests
- 📝 Verify Community Reports
- 📈 Emergency Response Management
